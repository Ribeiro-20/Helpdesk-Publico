# Como contribuir

Obrigado pelo teu interesse em contribuir para este projeto!

## Como começar

1. Faz um fork do repositório no GitHub.
2. Clona o teu fork para a tua máquina.
3. Cria uma branch para a tua feature/correção: `git checkout -b minha-feature`.
4. Faz commit das tuas alterações: `git commit -m 'feat: descrição da alteração'`.
5. Faz push para o teu fork: `git push origin minha-feature`.
6. Abre um Pull Request.

## Boas práticas
- Segue o padrão de commits (ex: feat, fix, chore).
- Escreve código limpo e documentado.
- Atualiza/Cria testes se aplicável.
- Descreve claramente o que alteraste no PR.

## Dúvidas
Abre uma issue ou discute no PR!
